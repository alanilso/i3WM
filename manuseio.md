# Comandos

- **$mod+q kill**
  - *Fechar o aplicativo*. 
- **$mod+d dmenu**

# Foco

  - *Abrir o Dmenu*.
- **$mod+Left = focus left**
  - *Focar à esquerda*.
- **$mod+Right = focus right**
  - *Focar à direita*.
- **$mod+Down = focus down**
  - *Focar a baixo*.
- **$mod+Up = focus up**
  - *Focar à cima*. 

# Mover Janelas com Foco

- **$mod+Shift+Left = move left**
  - *Mover a janela focada para à esquerda*.
